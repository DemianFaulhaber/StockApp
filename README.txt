FUNCIONES A IMPLEMENTAR ANTES DE PUBLICAR LA VERSIÖN ALPHA:
1-Creación de tickets y modificación de ventas a través de la pestaña de ventas.
2-Edición a gran escala para aumentar precios.
3-Reestructuarción de la pestaña para agregar productos (posible incorporación en la pestaña de busqueda).
4-Almacenamiento de información de ventas a largo plazo con discriminación ente efectivo y otros métodos.
FUNCIONES A IMPLEMENTAR EN LA BETA:
1-Auto facturación.
2-Apertura y cierre de cajas.
3-Sistema de descarga y/o impresión de las facturas.
4-Botón de vuelta atrás en caso de error al editar.
5-Fácil acceso para una modificación másiva de la base de datos a través de excel.
FUNCIONES A IMPLEMENTAR EN LA v1.0:
1-Calendario con sistema de avisos para ambar partes respecto a turnos.
2-Sistema de organización de delivery.
3-Sistema de almacenamiento para la información de pedidos y proveedores.
4-Implementar sistema opcional de usuarios para procurar la escabilidad.
